# RUDEDOG Landing Draft V2

ดราฟนี้ใช้ asset ที่คัดใหม่จาก `02_Selected_For_Web` โดย hero product เป็นหมวก `ALPHA CAP` และ copy อ้างอิงจาก `00_Brand_Documents` เพื่อให้ tone ตรงกับ RUDEDOG Brand Book / Voice Guide

## Product Mix

- ALPHA CAP: hero, cover, packshot color swatches
- PL-CLEAR: apparel support visual, packshot, carousel
- BAG-MOTION: product packshot / cover story
- Brand mascot: dog bow tie

## Notes

- สีดำถูกใช้เป็น accent เท่านั้น ไม่ใช้เป็นพื้นหลังหลักของเว็บ
- รูปส่วนใหญ่ถูก optimize เป็น WebP ใน `assets/images`; ชุด ALPHA ใช้ JPEG ย่อขนาดเพื่อให้ไฟล์เล็กและเปิดได้เสถียรใน draft
- Shopee links เป็น search URL พร้อม UTM organic
- เพิ่ม `robots.txt`, `sitemap.xml`, canonical, Open Graph และ JSON-LD สำหรับ SEO/AI search
